tradpunkter Workflow

tradpunkter Workflow is a signed and notarized macOS workflow application with built-in documentation and downloadable releases.

Workflow is an application for organizing work using documents, drawers, statuses, and relationships. ​The system can be used to manage many different kinds of work, such as reminders, projects, activities, cases, orders, or other document-based processes.

​Workflow is flexible. The structure of the system is defined by configuration, so different users can adapt it to different ways of working.

In Workflow, a workflow is a structured way of moving work from one stage to another.

​Work is stored in documents. Each document belongs to a drawer. The drawer defines how that type of document behaves, for example:

​• which statuses it uses

• which dates it tracks

• which actions are available

• whether it uses product rows

• how it is printed

​This allows the system to support both simple and more advanced ways of organizing work.
